# Extension Organizer

A cyberpunk-themed Chrome extension manager that lets you organize your extensions into custom groups.

## Features
- Pin and organize extensions into custom groups
- Toggle extensions on/off with a single click
- Cyberpunk aesthetic with neon colors and glowing effects
- Clean, minimal interface

## Installation
1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (top right)
3. Click "Load unpacked"
4. Select the `extension-manager` folder

## Usage
- Click the extension icon to open the organizer
- Click "+ GROUP" to create a new group
- Click on any extension to toggle it on/off
- Click "×" on a group header to delete the group
- Extensions not in any group appear in the "Default" group

## Permissions
- `management`: Required to list, enable, and disable extensions
