# বাংলা সমার্থক শব্দ কুইজ অ্যাপ

এটি একটি ডায়নামিক MCQ টেস্ট অ্যাপ্লিকেশন যা বাংলা সমার্থক শব্দের উপর ভিত্তি করে তৈরি।

## বৈশিষ্ট্য

- **র‍্যান্ডম প্রশ্ন জেনারেশন**: প্রতিবার নতুন প্রশ্ন সেট
- **কাস্টমাইজেবল**: প্রশ্নের সংখ্যা নির্বাচন করা যায় (৫-৫০)
- **স্মার্ট অপশন জেনারেশন**: একই গ্রুপ থেকে সঠিক উত্তর, অন্য গ্রুপ থেকে ভুল অপশন
- **রিয়েল-টাইম স্কোরিং**: তাৎক্ষণিক ফলাফল
- **রেসপন্সিভ ডিজাইন**: মোবাইল ও ডেস্কটপ উভয়ে কাজ করে

## কীভাবে চালাবেন

1. **প্রয়োজনীয় প্যাকেজ ইনস্টল করুন:**
   ```bash
   pip install -r requirements.txt
   ```

2. **অ্যাপ্লিকেশন চালান:**
   ```bash
   python app.py
   ```

3. **ব্রাউজারে যান:**
   ```
   http://localhost:4421
   ```

## ফাইল স্ট্রাকচার

```
├── app.py              # মূল Flask অ্যাপ্লিকেশন
├── questions.json      # সমার্থক শব্দের ডেটা
├── templates/
│   └── index.html      # ফ্রন্টএন্ড UI
├── requirements.txt    # Python dependencies
└── README.md          # এই ফাইল
```

## কীভাবে কাজ করে

1. **ডেটা স্ট্রাকচার**: `questions.json` ফাইলে সমার্থক শব্দগুলো গ্রুপ অনুযায়ী সাজানো
2. **প্রশ্ন জেনারেশন**: 
   - একটি গ্রুপ থেকে র‍্যান্ডম শব্দ নির্বাচন (প্রশ্নের জন্য)
   - একই গ্রুপ থেকে আরেকটি শব্দ (সঠিক উত্তর)
   - অন্য গ্রুপ থেকে ৩টি ভুল অপশন
3. **স্কোরিং**: সঠিক উত্তরের জন্য পয়েন্ট এবং শেষে শতকরা হিসাব

## কাস্টমাইজেশন

নতুন সমার্থক শব্দ যোগ করতে `questions.json` ফাইলে নতুন গ্রুপ যোগ করুন:

```json
{
  "group": "নতুন_গ্রুপ",
  "words": ["শব্দ১", "শব্দ২", "শব্দ৩"]
}
```

## পোর্ট পরিবর্তন

`app.py` ফাইলের শেষ লাইনে পোর্ট নম্বর পরিবর্তন করা যায়:
```python
app.run(debug=True, port=4421)  # এখানে পোর্ট নম্বর পরিবর্তন করুন
```
