##Study Notes Creation Instructions##

**Main Task**: Update the **1Main.md** content with new content from **Given.md** file

**Workflow Process**:
1. User adds new content to **Given.md**
2. Read and process content from **Given.md**
3. Integrate/adjust the content into **1Main.md** using proper formatting
4. After successful integration, **delete all content from Given.md**

**Purpose**: Create comprehensive study notes from provided content using custom markdown syntax

##Custom Markdown Syntax##

**Text Formatting**
- ##text## = **Big headings/titles**
- **text** = **Bold text for emphasis**
- @@text@@ = @@Italic text for special notes@@
- ==text== = **Black Background**

**List Formatting**
- text = Primary list items
-- text = Secondary list items (sub-points)
--- text = Tertiary list items (sub-sub-points)

##Content Organization Guidelines##

**Structure Requirements**
- Start with main title using ##title##
- Include **main topics** section with bold formatting
- Organize content into logical sections with ##section titles##
- Use appropriate list levels for hierarchical information

**Content Processing**
- Read provided content (from clipboard files or direct text)
- Extract key information and organize systematically
- Create clear categories and subcategories
- Include important facts, figures, and definitions
- Add exam-relevant information when applicable

**Formatting Rules**
- NO use of --- as section dividers (this is list level 3)
- Use ##headings## for major sections
- Use **bold** for important terms and concepts
- Use @@italic@@ for special notes or conclusions
- Maintain consistent list hierarchy
